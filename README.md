# PIX Distributed System

Sistema inspirado no PIX brasileiro desenvolvido com Java + Spring Boot.

## Tecnologias

- Java 21
- Spring Boot
- PostgreSQL
- Docker
- Spring Data JPA
- Spring Security
- Maven

## Objetivo

Simular transferências instantâneas entre contas utilizando arquitetura backend moderna.

## Funcionalidades atuais

- Cadastro de contas
- Transferência PIX
- Persistência PostgreSQL
- Dockerização do banco
